# Required Documents for TCET Admission

## Mandatory Documents
1. **10th Marksheet** - Original and verified photocopy
2. **12th Marksheet** - Original and verified photocopy
3. **Aadhar Card** - Front and back photocopy
4. **Passport Size Photos** - 4 recent photographs (white background)
5. **Caste Certificate** - Required for reserved category students (SC/ST/OBC/EWS)
6. **Domicile Certificate** - For Maharashtra domicile students
7. **MHT-CET / JEE Scorecard** - Original scorecard from entrance exam
8. **Transfer Certificate (TC)** - From previous institution
9. **Migration Certificate** - If from a different university/board
10. **Gap Certificate** - If there is a gap year after 12th

## Document Upload Process
- Go to the "Documents" section in the sidebar
- Select the document type from the dropdown
- Upload a clear scan or photo (PDF, JPG, PNG accepted)
- Maximum file size: 5MB per document
- Documents are verified within 2-3 working days
- You will receive a notification once verified

## Document Verification Status
- **Pending** - Document uploaded, awaiting review
- **Under Review** - Being checked by the admin team
- **Verified** - Document accepted and approved
- **Rejected** - Document unclear or incorrect, re-upload needed

## Tips for Document Upload
- Ensure all text is clearly readable
- Photos should be well-lit and not blurry
- PDF format is preferred for marksheets
- Keep original documents safe for physical verification during admission
